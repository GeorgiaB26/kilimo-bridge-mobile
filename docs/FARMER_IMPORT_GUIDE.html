Farmer Import File Guide · For cooperative administrators · July 2026

File format

FormatNotes
Excel (.xlsx)Preferred — upload the original workbook
CSV (.csv)Excel → File → Save As → CSV (Comma delimited)

One cooperative per file. Name the file after the cooperative (e.g. LEOART MACADAMIA FARMERS ASSOCIATION KENYA 2026.xlsx). If there is no Membership Group column, the file name is used as the cooperative name.

Required columns

Name — headers: Name, Farmer Name, Full Name, Name of Farmer

  • Letters, spaces, hyphens, apostrophes only
  • John Kamau ✓
  • John2 ✗   Mary & Sons ✗

Phone — headers: Phone, Mobile, Mobile No, Contact, Tel

  • One unique number per farmer
  • Kenya: 0712345678 or +254712345678
  • Format the phone column as Text in Excel (keeps the leading 0)

Recommended columns

ColumnExamples
District / CountyKirinyaga
Sub-CountyKirinyaga Central
Gender / SexM, F (blank → Other)
ID NumberOptional (blank → auto PENDING-ID)

Before you upload

  • Every farmer has a unique phone number
  • All phones are 07…… or +254…… format
  • Names contain letters only
  • Phone column formatted as Text in Excel
  • File saved as .xlsx or CSV (Comma delimited)
